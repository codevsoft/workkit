> This is an app to handle the clocked schedule of the workers in your company

### ✨ [Demo](https://www.codevsoft.dev)

## Install

```sh
yarn install
```

## Usage

```sh
npx nx serve workkit-fe

// in another terminal
npx nx serve workkit-be
```

## Run tests

```sh
npx nx affected:test
```
